Git Commands

git init

This command is used to start a new repository.

Usage: git init [repository name]

Example: git init https://github.com/Mounika-sdet-9/git-tutorial.git

git clone

This command is used to obtain a repository from an existing URL.

Usage: git clone [url]

Example: git clone https://github.com/Mounika-sdet-9/git-tutorial.git

git remote

This command is used to connect your local repository to the remote server.

Usage: git remote add [variable name] [Remote Server Link]

Example: git remote add origin https://github.com/Mounika-sdet-9/git-tutorial.git

git config

This command sets the author name and email address respectively to be used with your commits.

Usage: git config –global user.name “[name]”

Example: git config –global user.name "Mounika"

Usage: git config –global user.email “[email address]”

Example: git config –global user.name "mounika.sdet@gmail.com"

git pull

This command fetches and merges changes on the remote server to your working directory.

Usage: git pull [Repository Link]

Example: git pull https://github.com/Mounika-sdet-9/git-tutorial.git

git status

This command lists all the files that have to be committed.

Usage: git status

git diff

This command shows the file differences which are not yet staged.

Usage: git diff

git add

This command adds a file to the staging area.

Usage: git add [file]

Example: git add index.html

git commit

This command records or snapshots the file permanently in the version history.

Usage: git commit -m “[ Type in the commit message]”

Example: git commit -m "git initial commit"

git push

This command sends the committed changes of master branch to your remote repository.

Usage: git push [variable name] master

Example: git push origin master

git branch

This command lists all the local branches in the current repository.

Usage: git branch

This command creates a new branch.

Usage: git branch [branch name]

Example: git branch newBranch

git merge

git stash

This command temporarily stores all the modified tracked files.

Usage: git stash save
